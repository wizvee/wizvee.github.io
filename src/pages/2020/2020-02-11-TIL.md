## Clean Code for JavaScript

### Method Chaining

```javascript
class Car {
  constructor() {
    this.make = 'KIA';
    this.color = 'white';
  }
  setMake(make) {
    this.make = make;
    // 메모: 체이닝을 위해 this를 리턴합니다.
    return this;
  }
  setColor(color) {
    this.color = color;
    // 메모: 체이닝을 위해 this를 리턴합니다.
    return this;
  }
  save() {
    console.log(this.make, this.color);
    // 메모: 체이닝을 위해 this를 리턴합니다.
    return this;
  }

  const car = new Car()
    .setColor('pink')
    .setMake('Ford')
    .save();
}
```
